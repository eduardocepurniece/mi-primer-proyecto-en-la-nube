# mi-primer-proyecto-en-la-nube
Proyecto Final RA2: “Mi Primer Proyecto en la Nube” 

Descripción del proyecto 
El participante desarrollará un proyecto básico en su entorno local y lo gestionará utilizando Git, para posteriormente publicarlo en un repositorio remoto en GitHub. 
El objetivo es aplicar de manera integrada todos los conocimientos adquiridos durante la unidad, desde la creación del repositorio hasta la sincronización de cambios.
